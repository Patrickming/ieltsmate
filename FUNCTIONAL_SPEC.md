## 〇、完整功能清单

> 以下是本应用的**所有功能**的完整列表，按功能领域分组。

### A. 笔记管理

| # | 功能 | 说明 | 出现位置 | 状态 |
|---|------|------|----------|------|
| A1 | 添加笔记 | 输入英文内容（如"get out of — 避免"），选择分类方式后保存 | 仪表盘"添加新笔记"按钮、知识库"添加笔记"按钮 → 弹出 QuickNoteModal | 🎨 保存动画有效，但笔记不会真正写入列表（无后端） |
| A2 | 分类方式选择 | 添加笔记时可选"AI 自动识别"或"手动选择"分类 | QuickNoteModal 内 | 🎨 AI 识别是模拟的（`detectCategory` 启发式匹配关键词），手动选择可用 |
| A3 | AI 分类预览 | 输入内容后自动展示 AI 识别出的分类和内容预览，可确认或修改 | QuickNoteModal 内 | 🎨 纯前端模拟 |
| A4 | 浏览全部笔记 | 以 3 列卡片网格展示所有杂笔记（翻转卡片），支持筛选和搜索 | 知识库页 `/kb` | ✅ |
| A5 | 按大分类筛选 | Tab 切换：全部 / 杂笔记 / 写作 / 收藏夹，切换后内容立即变化 | 知识库页 分组 Tab 栏 | ✅ |
| A6 | 按子分类筛选 | 选了"杂笔记"后出现：口语/短语/句子/同义替换/拼写/单词；选了"写作"后出现：全部/大作文/小作文 | 知识库页 子分类 Pills | ✅ |
| A7 | 搜索笔记（知识库内） | 在知识库页面顶部的搜索框中输入关键词，实时过滤笔记（匹配英文内容或中文翻译） | 知识库页 搜索框 | ✅ 杂笔记搜英文+中文，写作搜文件名 |
| A8 | 搜索笔记（全局快捷） | Cmd/Ctrl+K 弹出全局搜索弹窗，输入关键词搜索，上下键导航，Enter 跳转详情 | 全局 SearchModal | ✅ |
| A9 | 查看笔记详情 | 点击卡片进入详情页，展示英文内容、中文翻译、音标、AI 延伸内容（同义词/反义词/例句/记忆技巧） | 笔记详情页 `/kb/:id` | ✅ |
| A10 | 添加个人备注 | 在笔记详情页可以添加自己的文字备注，支持多条 | 笔记详情页 "我的备注" 区域 | 🎨 仅保存在页面局部状态，刷新后丢失 |
| A11 | 编辑笔记 | 笔记详情页有"编辑"按钮 | 笔记详情页 操作卡片 | 🚧 按钮存在但无 onClick |
| A12 | 删除笔记 | 笔记详情页有"删除"按钮 | 笔记详情页 操作卡片 | 🚧 按钮存在但无 onClick |
| A13 | 上/下一条导航 | 在笔记详情页底部可以直接跳到上一条或下一条笔记 | 笔记详情页 底部导航 | ✅ |
| A14 | 滚动位置记忆 | 从知识库点进详情再返回时，自动恢复之前的滚动位置 | 知识库 + NoteCard | ✅ sessionStorage 实现 |
| A15 | 从侧边栏快速跳转分类 | 侧边栏点击"口语"→ 知识库自动筛选到杂笔记+口语 | 侧边栏 分类区域 | ✅ 通过 URL 参数 `?group=杂笔记&cat=口语` |

| D5 | 实时预览卡片数量 | 底部固定栏显示当前筛选条件下有多少张卡片可以复习，0 张时按钮禁用 | 复习选择页 底部栏 | ✅ |

| D10 | 评分记得/不记得 | 翻面后出现"😞 不记得"和"😊 记得"按钮，点击后记录评分并自动跳到下一张 | 复习卡片页 | ✅ |

| D14 | 复习进度显示 | 顶部显示当前卡片序号/总数 + 紫色进度条 | 复习卡片页 | ✅ |

## 十三、收藏系统

收藏功能贯穿整个应用，出现在三个地方：

### 1. NoteCard 卡片（知识库、收藏夹、仪表盘）
- 位置：卡片翻面后的右下角 footer
- 样式：compact 变体 — 仅一个小心形图标
- 点击不触发卡片导航

### 2. 笔记详情页
- 位置：右侧操作卡片内，编辑/删除按钮下方
- 样式：full 变体 — 心形图标 + 文字标签
- 有滑动文字切换动画

### 3. 复习进行中页面
- 位置：评分按钮旁边（始终可见）
- 样式：Pill 按钮变体 — 透明边框按钮 + 悬停填充动画
- 已收藏时边框和颜色变红

### 收藏数据

- 存储在 `localStorage`（键 `ielts-favorites`），是笔记 ID 的 JSON 数组
- 收藏/取消收藏是即时的，所有页面实时同步（通过 Zustand 全局状态）
- 知识库的"收藏夹"Tab 和复习选择的"收藏夹"来源都读取这个数据

## 十四、动画与过渡效果汇总

| 场景 | 动画类型 | 描述 |
|------|----------|------|
| 侧边栏导航切换 | layoutId 共享布局动画 | 激活背景在不同导航项之间平滑滑动 |
| 页面切换 | 淡入过渡 | 新页面从透明到可见 |
| NoteCard 翻转 | CSS 3D rotateY | 悬停时卡片 180° 翻转，500ms |
| NoteCard 光束 | CSS @keyframes | 反面边框有彩色光束不停旋转（4s 一圈） |
| NoteCard 光球 | CSS @keyframes | 两个装饰光球上下浮动（2.6-3s 循环） |
| 复习翻卡 | Framer Motion rotateY | 点击/空格触发 180° 翻转，420ms 弹性缓动 |
| 卡片列表出现 | stagger 延迟淡入 | 多张卡片依次从下方滑入，每张间隔 30-40ms |
| 弹窗出现 | scale + opacity | 从 0.95 缩放到 1 + 透明到不透明，弹簧动画 |
| 弹窗遮罩 | opacity | 黑色遮罩淡入淡出 |
| AI 面板 | translateX | 从右侧滑入，弹簧动画 |
| 收藏心形 | @keyframes heartPulse | 1→1.4→1→1.3→1 脉冲缩放 |
| 收藏文字切换 | translateY + opacity | "收藏"上移消失，"已收藏"从下方滑入 |
| Pill 按钮 | ::before scale | 中心圆形从 0 扩展到 3 倍，填满按钮 |
| Pill 按钮 | scale | 悬停时按钮放大 1.08 倍 |
| 进度条 | width 过渡 | 紫色/绿色填充带 0.4s ease-out |
| 复习总结数字 | spring 物理动画 | 数字从 0 弹性递增到目标值 |
| 热力图今日格子 | 颜色切换 | 任务全完成时从紫色变金色 |
| 子分类展开 | height + opacity | AnimatePresence 自动计算高度变化 |
| 退出确认弹窗 | scale + opacity | 从 0.92 缩放弹出 |
| 底部固定栏 | translateY | 从下方 20px 弹入，延迟 0.4s |
| 卡片数字变化 | scale bounce | 数字变化时先放大 1.3 倍再缩回 |
