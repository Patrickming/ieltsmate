# 安全漏洞报告

若你发现 IELTSmate 中存在**可被利用的安全漏洞**（例如未授权访问、注入、敏感数据泄露等），请**不要**在公开 Issue 中披露细节。

## 如何报告

1. 在 GitHub 仓库使用 **Security → Report a vulnerability**（若已开启私有安全建议），或  
2. 向仓库维护者发送私信说明（通过你在仓库主页提供的联系方式，若有）。

请尽量包含：

- 受影响版本或分支、复现步骤、潜在影响范围  
- 如有可能，附最小复现或修复思路（非必须）

维护者会在合理时间内查看并回复。在修复发布前，请避免对外公开可利用细节。

**说明**：本仓库为自托管学习类应用示例；部署到公网时请务必自行加固（HTTPS、密钥管理、数据库访问控制等）。
