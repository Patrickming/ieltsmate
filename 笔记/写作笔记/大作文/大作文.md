### 主体段中心句的编写逻辑

> ABC的逻辑链 

#### 从A出发

**优缺点/观点类**（目的-导致...）



因（题目）到果  A -> C

**同意：** **A** `征收拥堵税` **B** 政府收入增加 **C** 有更多钱改善交通

**不同意：**A `征收拥堵税` **B** 用车成本增高，经济负担加重 **C** 生活水平降低

![image-20250630232602163](./assets/image-20250630232602163.png)

#### 从c出发

**观点类** （结果-因为...）



果（题目）到因  C -> A

**同意：** **A** 汽车越来越便宜 **B** 大家都开车，拥堵严重 **C** `应该`征收拥堵税

**不同意：**A汽车是生活必需品 **B** 收拥堵税对生活造成不便 **C** `不应该`征收拥堵税

![image-20250630232626652](./assets/image-20250630232626652.png)

![image-20250701005736270](./assets/image-20250701005736270.png)

#### ABC的连接编写

中心句式：



观点段（第二段）

中心句



**A-> B-> C**

A **will cause/ lead to/ ** **can** **increase**(improve)/ **can** **be beneficial to**  + B... **and/，which can/making** + C 

`A will lend to B, which can `



**A ->C 因为 B**

A **has made/ can make** + C **because/ as a result of** + B

`A can make C as a result of B`



**As** A , C because/ as a result of + B
